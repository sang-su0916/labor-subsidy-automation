# 고용지원금 자동분석 서비스 사용 가이드

## 접속 방법

1. 웹브라우저에서 접속: **https://goyoung-subsidy.vercel.app**
2. 비밀번호 입력: `goyoung2026`
3. "입장하기" 버튼 클릭

---

## 사용 방법

### 방법 1: 서류 업로드 (자동 분석)

1. 메인 화면에서 **"서류 업로드로 시작"** 클릭
2. 근로자의 서류를 업로드
   - 주민등록등본
   - 근로계약서
   - 4대보험 가입내역 등
3. 시스템이 자동으로 정보 추출
4. 추출된 정보 확인 후 **"다음"** 클릭
5. 지원 가능한 고용지원금 목록 확인
6. 리포트 다운로드 (PDF)

### 방법 2: 직접 입력

1. 메인 화면에서 **"직접 입력하기"** 클릭
2. 근로자 정보 직접 입력
   - 이름, 생년월일
   - 고용 형태, 입사일
   - 근무 시간, 급여 등
3. **"분석하기"** 클릭
4. 지원 가능한 고용지원금 확인
5. 리포트 다운로드

---

## 지원 가능한 고용지원금 종류

| 지원금 | 대상 |
|--------|------|
| 청년일자리도약장려금 | 만 15~34세 청년 채용 시 |
| 고용촉진장려금 | 취업 취약계층 채용 시 |
| 고용유지지원금 | 고용 유지 노력 기업 |
| 고령자계속고용장려금 | 60세 이상 재고용 시 |
| 고령자고용지원금 | 고령자 신규 채용 시 |
| 출산육아기 고용안정장려금 | 육아휴직 대체인력 채용 시 |
