## 이 앱은 무엇인가요?

회사에서 직원을 고용하면 정부에서 다양한 지원금을 받을 수 있습니다. 하지만 어떤 지원금을 받을 수 있는지, 얼마나 받을 수 있는지 알아보기가 복잡하죠.

**이 앱은 회사 서류를 업로드하면 자동으로:**
- ✅ 받을 수 있는 지원금 종류를 찾아줍니다
- ✅ 예상 지원 금액을 계산해 줍니다
- ✅ 신청에 필요한 서류 목록을 알려줍니다
- ✅ PDF 보고서로 다운로드할 수 있습니다

---

## 시작하기

### 필요한 것

- **Node.js** 18버전 이상
- **npm** 9버전 이상

### 설치 방법

```bash
# 1. 프로젝트 폴더로 이동
cd Labor-test

# 2. 필요한 파일 설치
npm install
```

### 실행 방법

```bash
# 앱 실행 (백엔드 + 프론트엔드 동시 실행)
npm run dev
```

실행 후:
- 📱 **웹 화면**: http://localhost:5173 에서 확인
- 🖥️ **서버**: http://localhost:3010 에서 동작

---

## 사용 방법

### 전체 흐름

```
① 서류 업로드  →  ② 데이터 추출  →  ③ 지원금 확인  →  ④ 보고서 다운로드
```

---

### 1단계: 서류 업로드

**화면 위치**: 홈 → "서류 업로드 시작" 버튼 클릭

#### 업로드할 서류

| 서류 종류 | 설명 | 파일 형식 |
|----------|------|----------|
| 사업자등록증 | 회사 기본 정보 확인용 | PDF, 이미지 |
| 임금대장 | 직원 목록, 급여, 입사일 확인용 | Excel, PDF |
| 근로계약서 | 고용 형태, 근무 시간 확인용 | PDF, Word |
| 4대보험 가입자명부 | 보험 가입 여부 확인용 | Excel, PDF |

#### 업로드 방법

1. **파일 끌어다 놓기** - 파일을 업로드 영역에 드래그
2. **클릭해서 선택** - 업로드 영역 클릭 후 파일 선택

#### 문서 유형 선택

파일을 올리면 각 파일이 어떤 서류인지 선택해야 합니다:

```
📄 회사서류.pdf  →  [사업자등록증 ▼] 선택
📊 급여명세.xlsx →  [임금대장 ▼] 선택
```

> 💡 **팁**: 모든 파일의 유형을 선택해야 다음 단계로 넘어갈 수 있어요!

---

### 2단계: 데이터 추출

**자동으로 진행됩니다**

업로드한 서류에서 AI가 필요한 정보를 자동으로 읽어냅니다:

- 회사명, 사업자번호
- 직원 이름, 주민번호, 입사일, 급여
- 4대보험 가입 여부
- 고용 형태 (정규직/계약직/시간제)

#### 추출되는 직원 정보

```
예시:
┌──────────┬─────────┬────────────┬───────────┐
│ 이름     │ 나이    │ 입사일     │ 월급여    │
├──────────┼─────────┼────────────┼───────────┤
│ 김철수   │ 30세    │ 2023-03-15 │ 350만원   │
│ 이영희   │ 24세    │ 2024-01-02 │ 220만원   │
│ 최고령   │ 60세    │ 2010-01-15 │ 650만원   │
└──────────┴─────────┴────────────┴───────────┘
```

> ⏳ 문서 양에 따라 몇 초~몇 분이 소요될 수 있습니다

---

### 3단계: 지원금 확인

**화면에서 확인할 수 있는 것들:**

#### 총 예상 지원금

```
┌─────────────────────────────────┐
│     총 예상 지원금              │
│                                 │
│     ★ 14,400,000원 ★           │
│                                 │
│     3개 프로그램 지원 가능      │
└─────────────────────────────────┘
```

#### 프로그램별 상세 내역

각 프로그램마다:
- 지원 가능 여부 (✅ 가능 / ❌ 불가)
- 예상 지원 금액
- 계산 방법 (월 00만원 × 00개월)
- 참고 사항

---

### 4단계: 보고서 다운로드

**4가지 종류의 보고서를 다운로드할 수 있습니다:**

| 보고서 종류 | 내용 | 용도 |
|------------|------|------|
| **PDF 보고서** | 분석 결과 요약 | 내부 검토용 |
| **신청 체크리스트** | 프로그램별 필요 서류, 신청처, 기한 | 신청 준비용 |
| **상세 분석 보고서** | 직원별 분석, 계산 내역, 지급 일정 | 상세 검토용 |
| **신청서 작성 보조 자료** | 직원 명부, 대상자 목록 정리 | 노무사/인사담당자용 |

#### 다운로드 버튼 위치

```
┌─ 기본 보고서 ─────────────────────────┐
│ [PDF 보고서 다운로드]                 │
│ [신청 체크리스트 다운로드]            │
└───────────────────────────────────────┘

┌─ 상세 분석 (인디고색) ────────────────┐
│ [상세 분석 보고서 다운로드]           │
└───────────────────────────────────────┘

┌─ 신청 보조 (에메랄드색) ──────────────┐
│ [신청서 작성 보조 자료 다운로드]      │
└───────────────────────────────────────┘
```

## 지원금 프로그램 안내

### 1. 청년일자리도약장려금 💙

**대상**: 15~34세 청년을 정규직으로 채용한 기업

| 항목 | 내용 |
|------|------|
| 지원금액 | 월 60만원 × 12개월 = **720만원** |
| 추가 인센티브 | 비수도권 기업 최대 720만원 추가 |
| 주요 조건 | 6개월 이상 고용유지, 4대보험 가입 |
| 신청 시기 | 6개월 고용유지 후 신청 |

---

### 2. 고용촉진장려금 💚

**대상**: 취업취약계층(장애인, 고령자, 경력단절여성 등)을 채용한 기업

| 항목 | 내용 |
|------|------|
| 지원금액 | 월 30~60만원 × 최대 24개월 |
| 최대 금액 | **1,440만원** |
| 주요 조건 | 취업지원프로그램 이수자 채용 |

---

### 3. 고용유지지원금 🧡

**대상**: 경영 악화로 휴업/휴직하며 고용을 유지하는 기업

| 항목 | 내용 |
|------|------|
| 지원금액 | 휴업수당의 1/2~2/3 |
| 1일 한도 | 최대 66,000원 |
| 연간 한도 | 180일 |
| 주요 조건 | 매출 15% 이상 감소 증빙 필요 |

---

### 4. 고령자계속고용장려금 💜

**대상**: 정년 연장/폐지 또는 재고용 제도를 도입한 기업

| 항목 | 수도권 | 비수도권 |
|------|--------|----------|
| 지원금액 | 분기 90만원 (월 30만원) | **분기 120만원 (월 40만원)** |
| 최대 금액 (3년) | 1,080만원 | **1,440만원** |
| 주요 조건 | 60세 이상 근로자 계속 고용 | 60세 이상 근로자 계속 고용 |

> 📌 **2026년 변경사항**: 비수도권 사업장은 월 40만원(분기 120만원)으로 인상되었습니다.

---

### 5. 고령자고용지원금 💜

**대상**: 60세 이상 근로자 수가 증가한 기업

| 항목 | 내용 |

## 자주 묻는 질문

### Q. 어떤 파일 형식을 지원하나요?

**지원 형식:**
- PDF (.pdf)
- Excel (.xlsx, .xls)
- Word (.docx)
- 이미지 (.jpg, .png)

**파일 크기:** 최대 50MB

---

### Q. 데이터 추출이 정확하지 않아요

서류 상태에 따라 추출 정확도가 달라질 수 있습니다:

| 상태 | 정확도 |
|------|--------|
| 깨끗한 스캔/원본 파일 | ⭐⭐⭐ 높음 |
| 사진 촬영 파일 | ⭐⭐ 보통 |
| 손글씨/흐린 문서 | ⭐ 낮음 |

> 💡 **팁**: 가능하면 원본 파일이나 고화질 스캔본을 사용하세요!

---

### Q. 앱이 실행되지 않아요

**확인할 것들:**

1. Node.js 버전 확인
```bash
node --version  # v18.0.0 이상이어야 함
```

2. 의존성 재설치
```bash
npm install
```

3. 포트 충돌 확인
```bash
# 3010 또는 5173 포트를 사용 중인 프로그램이 있는지 확인
lsof -i :3010
lsof -i :5173
```
