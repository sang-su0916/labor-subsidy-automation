## 1. 이 앱은 무엇인가요?

회사에서 직원을 고용하면 정부에서 다양한 **고용지원금**을 받을 수 있습니다.
하지만 어떤 지원금을 받을 수 있는지, 얼마나 받을 수 있는지 알아보기가 복잡하죠.

**이 앱은 회사 서류를 업로드하면 자동으로:**

- 받을 수 있는 **지원금 6개 프로그램**을 분석합니다
- **예상 지원 금액**을 계산합니다
- 프로그램별 **대상 직원**을 자동으로 찾아줍니다
- 신청에 필요한 **서류 목록**을 정리해줍니다
- 전문가용 **PDF 보고서 3종**을 다운로드할 수 있습니다

### 두 가지 사용 방법

| 방법 | 설명 | 추천 대상 |
|------|------|-----------|
| **서류 업로드** | 사업자등록증, 임금대장 등을 올리면 AI가 자동 분석 | 실제 서류가 준비된 사업장 |
| **직접 입력** | 회사 정보와 직원 정보를 직접 타이핑 | 서류 없이 빠르게 확인하고 싶은 분 |

## 2. 접속 방법

### 웹 브라우저로 접속

1. 브라우저(크롬, 사파리 등)를 엽니다
2. 주소창에 입력합니다: **https://goyoung-subsidy.vercel.app**
3. 비밀번호를 입력합니다: `goyoung2026`
4. "확인" 버튼을 누르면 메인 화면이 나타납니다

> 설치가 필요 없습니다. PC, 태블릿, 스마트폰 모두 사용 가능합니다.

### 메인 화면 구성

메인 화면에는 다음이 표시됩니다:

- **상단 메뉴**: 홈 / 서류 업로드 / 지원금 확인
- **시작 버튼 2개**: "서류 업로드" / "직접 입력"
- **6개 지원금 프로그램 카드**: 각 프로그램의 지원금액과 주요 요건
- **필요 서류 목록**: 펼쳐서 확인 가능
- **이용 방법 안내**: 3단계 프로세스 설명

## 3. 방법 A: 서류 업로드로 분석하기 (추천)

> **전체 흐름**: 서류 업로드 → AI 데이터 추출 → 지원금 분석 → 보고서 다운로드

### 3-1. 준비할 서류

| 순서 | 서류 이름 | 왜 필요한가요? | 파일 형식 | 필수 여부 |
|:----:|----------|---------------|-----------|:---------:|
| 1 | **사업자등록증** | 회사명, 사업자번호, 대표자, 소재지 확인 | PDF, 이미지 | **필수** |
| 2 | **임금대장** (또는 급여명세서) | 직원 목록, 급여, 입사일, 주민번호 확인 | **Excel 추천**, PDF | **필수** |
| 3 | **근로계약서** | 고용형태, 근무시간, 계약기간 확인 | PDF, Word | 선택 |
| 4 | **4대보험 가입자명부** | 고용보험 가입일, 퇴사자, 보험 상태 확인 | **Excel 추천**, PDF | **강력 권장** |

> **파일 제한**: 최대 100MB / 한 번에 최대 150개 파일까지 업로드 가능
>
> **지원 형식**: PDF, Excel(.xlsx, .xls), Word(.docx), 이미지(PNG, JPG, JPEG, WEBP)

> **임금대장은 Excel 파일이 가장 정확합니다.** Excel은 직접 파싱하여 100% 정확도를 보장하고, PDF는 AI/OCR로 읽기 때문에 간혹 오류가 있을 수 있습니다.

### 3-2. 서류 업로드하기

1. 메인 화면에서 **"서류 업로드"** 버튼을 클릭합니다
2. 업로드 화면이 나타납니다

#### 문서 유형을 먼저 선택하세요

화면 상단에 4개 버튼이 있습니다:

```
[🏢 사업자등록증]  [💰 임금대장/급여명세서]  [📋 근로계약서]  [🛡️ 4대보험 가입자명부]
```

**업로드할 서류에 맞는 버튼을 먼저 클릭**한 뒤, 파일을 올려주세요.

> 예시:
> - 사업자등록증 PDF를 올릴 때 → "🏢 사업자등록증" 클릭 → 파일 업로드
> - 급여 Excel을 올릴 때 → "💰 임금대장/급여명세서" 클릭 → 파일 업로드

#### 파일 올리는 방법 (2가지)

- **방법 1**: 파일을 업로드 영역에 **끌어다 놓기** (드래그 앤 드롭)
- **방법 2**: 업로드 영역을 **클릭**하여 파일 탐색기에서 선택

#### 업로드 후 확인

파일이 올라가면 아래에 목록이 나타납니다:

```
📄 사업자등록증_가을식품.pdf   (450KB)   [사업자등록증]   [삭제]
📊 2025년_임금대장.xlsx       (1.2MB)   [임금대장]       [삭제]
📋 근로계약서_김정민.pdf      (320KB)   [근로계약서]     [삭제]
🛡️ 4대보험_가입자명부.xlsx   (890KB)   [4대보험]        [삭제]
```

- 문서 유형이 잘못 지정되었으면 **옆의 드롭다운**에서 변경 가능
- 잘못 올린 파일은 **삭제** 버튼으로 제거
- **모든 파일에 문서 유형이 지정되어야** 다음 단계로 진행 가능

### 3-3. 데이터 추출 (자동)

모든 파일 업로드가 끝나면 **"데이터 추출하기"** 버튼을 클릭합니다.

AI가 자동으로 서류를 읽고 필요한 정보를 추출합니다:

```
📊 전체 진행률: ████████████████░░░░  80%

📄 사업자등록증_가을식품.pdf      ✅ 완료
📊 2025년_임금대장.xlsx           ✅ 완료
📋 근로계약서_김정민.pdf          🔄 진행 중...
🛡️ 4대보험_가입자명부.xlsx       ⏳ 대기 중
```

> 문서 양에 따라 몇 초 ~ 수 분 소요될 수 있습니다.

#### 추출되는 정보

| 서류 | 추출 항목 |
|------|----------|
| 사업자등록증 | 상호명, 사업자번호, 대표자명, 소재지, 업종, 개업일 |
| 임금대장 | 직원 이름, 주민번호, 입사일, 월급여, 고용형태 |
| 근로계약서 | 계약기간, 근무시간, 급여, 4대보험 가입 여부 |
| 4대보험 가입자명부 | 고용보험 취득일, 상실일, 상실사유, 4대보험 가입 상태 |

#### 추출 결과 확인

추출이 끝나면 결과를 확인할 수 있습니다.

각 항목 옆에 **신뢰도**가 표시됩니다:

- 🟢 **높음**: AI가 확실하게 읽은 정보
- 🟡 **보통**: 확인이 필요할 수 있는 정보
- 🔴 **낮음**: 잘못 읽었을 가능성이 있는 정보

> 추출 결과가 정확한지 간단히 확인하고, 이상 없으면 **"지원금 확인하기"** 버튼을 눌러주세요.

## 4. 방법 B: 직접 입력으로 간단히 확인하기

> 서류가 없거나, 빠르게 예상 금액만 확인하고 싶을 때 사용합니다.
>
> **전체 흐름**: 사업장 정보 → 직원 정보 → 계좌 정보 → 분석 결과

### 4-1단계: 사업장 정보 입력

메인 화면에서 **"직접 입력"** 버튼을 클릭합니다.

입력 항목:

| 항목 | 예시 | 필수 여부 |
|------|------|:---------:|
| **상호명** | 주식회사 가을식품 | **필수** |
| 사업자등록번호 | 654-81-01412 | 선택 |
| 대표자명 | 박노철 | 선택 |
| 개업일 | 2018-03-15 | 선택 |
| 사업장 소재지 | 경기도 고양시... | 선택 |
| **지역 구분** | 수도권 / 비수도권 | **필수** |
| **기업 규모** | 중소기업 / 대기업 | **필수** |
| 고용보험 관리번호 | 12345678-01 | 선택 |
| 업종코드 | J62 | 선택 |

> **지역 구분이 중요합니다!** 비수도권은 청년일자리도약장려금 인센티브와 고령자계속고용장려금 금액이 더 높습니다.

입력 후 **"다음: 직원 정보 입력 →"** 버튼을 클릭합니다.

### 4-2단계: 직원 정보 입력

**"+ 직원 추가"** 버튼을 눌러 직원을 한 명씩 등록합니다.

각 직원별 입력 항목:

| 항목 | 예시 | 필수 여부 | 설명 |
|------|------|:---------:|------|
| **이름** | 김정민 | **필수** | |
| 주민등록번호 | 650101-1234567 | 선택 | 입력하면 생년월일 자동 계산 |
| **생년월일** | 1965-01-01 | **필수** | 주민번호 입력 시 자동 입력됨 |
| **입사일** | 2020-03-01 | **필수** | 고용기간 계산에 사용 |
| 고용보험 가입일 | 2020-03-01 | 선택 | |
| 직위/직책 | 생산부장 | 선택 | |
| 부서 | 생산팀 | 선택 | |
| 월급여 | 3000000 | 선택 | 최저보수 기준(124만원) 충족 여부 확인 |
| **고용형태** | 정규직/계약직/파트타임 | **필수** | 청년장려금은 정규직만 해당 |
| 고용보험 가입 | 체크/해제 | 기본 체크 | |
| 국민연금 가입 | 체크/해제 | 기본 체크 | |

> **주민등록번호를 입력하면 편합니다**: 생년월일이 자동으로 계산되고, 나이(청년/고령자) 판단이 정확해집니다. 입력 시 형식 검증도 자동으로 됩니다.

직원을 모두 입력한 뒤 **"다음: 계좌 정보 →"** 버튼을 클릭합니다.

### 4-3단계: 계좌 정보 입력 (선택)

지원금 수령 계좌를 입력합니다. **이 단계는 건너뛸 수 있습니다.**

| 항목 | 예시 |
|------|------|
| 은행명 | 국민은행 |
| 계좌번호 | 123456-78-901234 |
| 예금주 | (주)가을식품 |
| 계좌 유형 | 법인 명의 / 대표자 명의 |

입력 후 **"지원금 분석하기"** 버튼을 클릭합니다.

## 5. 분석 결과 보는 법

### 총 예상 지원금

화면 상단에 가장 큰 글씨로 표시됩니다:

```
┌──────────────────────────────────┐
│        총 예상 지원금             │
│                                  │
│     ★ 14,400,000원 ★            │
│                                  │
│     2개 프로그램 지원 가능        │
└──────────────────────────────────┘
```

### 데이터 품질 경고 (서류 업로드 시)

서류에서 추출한 데이터에 문제가 있으면 경고가 표시됩니다:

- 🔴 **높음**: 금액 계산에 영향을 미치는 중요한 누락 (예: 주민번호 없음 → 나이 계산 불가)
- 🟡 **보통**: 확인이 필요한 항목 (예: 급여 정보 불일치)
- 🔵 **낮음**: 참고사항

각 경고에는 **"💡 해결 방법"** 이 함께 표시됩니다.

### 문서 매칭 결과 (서류 업로드 시)

여러 서류를 올렸을 때, 서류 간 직원 정보가 잘 연결되었는지 보여줍니다:

```
┌─────────────────────────────────────────────┐
│  급여대장 직원: 19명   매칭 성공: 15명       │
│  청년(15~34세): 3명    고령자(60세+): 1명    │
│                                              │
│  매칭률: 79%                                 │
│                                              │
│  ⚠️ 4명이 근로계약서와 매칭되지 않았습니다.   │
│     퇴사자이거나 근로계약서가 누락된 경우     │
└─────────────────────────────────────────────┘
```

### 프로그램별 분석 결과

6개 프로그램 각각에 대해 결과가 표시됩니다:

#### 결과 상태 3가지

| 상태 | 의미 | 표시 |
|------|------|------|
| ✅ **지원 가능** | 대상자가 확인되고 요건 충족 | 초록색 + 예상 금액 |
| 🔍 **검토 필요** | 자동 확인 불가, 수동 확인 필요 | 노란색 + 안내 사항 |
| ❌ **지원 불가** | 대상자 없음 또는 요건 미충족 | 회색 + 사유 |

> **"검토 필요"란?**
> 시스템이 자동으로 판단할 수 없는 요건이 있는 경우입니다.
> 예를 들어, 고용촉진장려금은 "취업취약계층"을 채용해야 하는데, 장애인/경력단절여성 여부는 서류만으로 확인할 수 없습니다.
> **0원으로 표시되더라도 노무사가 확인하면 지원 가능할 수 있습니다.**

#### 각 프로그램 카드에서 확인할 수 있는 정보

- 프로그램 이름
- 지원 가능 여부
- 예상 금액 (월 ○○만원 × ○○개월, 또는 분기 ○○만원 × ○○분기)
- 대상 직원 수와 이름
- 충족된 요건 / 미충족 요건
- 신청 가능 시점 (입사 6개월 미만인 직원의 경우)
- 감원방지의무 위반 여부 (4대보험 명부 제출 시)

### 직원-프로그램 매트릭스 (직접 입력 시)

직접 입력 방식에서는 **어떤 직원이 어떤 프로그램에 해당하는지** 한눈에 볼 수 있는 표가 제공됩니다:

```
┌──────────┬───────────┬─────────┬──────────┐
│ 직원     │ 청년도약  │ 고령계속 │ 고령지원 │
├──────────┼───────────┼─────────┼──────────┤
│ 이수진   │ ✅ 720만  │         │          │
│ 김정민   │           │ ✅1,080 │ ✅ 240만 │
│ 박영호   │           │         │          │
└──────────┴───────────┴─────────┴──────────┘
```

### 중복 수급 제외 안내

한 직원이 동시에 받을 수 없는 프로그램이 있으면 자동으로 제외됩니다:

| 중복 조합 | 우선 적용 | 제외되는 프로그램 |
|-----------|----------|------------------|
| 청년일자리도약 + 고용촉진 | 청년일자리도약장려금 | 고용촉진장려금 |

## 8. 4대보험 가입자명부의 중요성

**4대보험 가입자명부를 함께 업로드하면 분석 정확도가 크게 올라갑니다.**

### 있을 때 vs 없을 때 비교

| 분석 항목 | 명부 **없을 때** | 명부 **있을 때** |
|----------|-----------------|-----------------|
| 고용보험 가입 여부 | 임금대장 기준 추정 | **실제 가입 상태 확인** |
| 재직/퇴사 구분 | 임금대장만으로 판단 | **상실일·상실사유로 정확 판단** |
| 감원방지의무 위반 확인 | **확인 불가** | **자동 감지** (권고사직/해고 이력) |
| 피보험자 수 | 추정치 | **정확한 수** |
| 고용보험 가입기간 | **확인 불가** | **취득일 기준 정확 계산** |
| 지원금 정확도 | "최대 예상" 수준 | **"실제 수령 가능"에 가까움** |

### 왜 중요한가요?

1. **감원방지의무 자동 검증**: 채용 전 1개월 내 권고사직(코드23)/해고(코드26) 이력이 있으면 청년일자리도약장려금을 받을 수 없는데, 이걸 자동으로 체크합니다
2. **퇴사자 정확 분리**: 상실일이 있는 직원은 자동 제외되어 현재 재직자만 정확 계산
3. **고용보험 가입 확인**: 대부분의 지원금이 "고용보험 가입" 요건이 있는데, 실제 데이터로 검증

> **결론**: 4대보험 명부를 넣으면 금액이 올라가거나(추가 대상자 발견) 내려갈 수(감원위반·미가입 발견) 있지만, **실제에 더 가까운 정확한 금액**을 알 수 있습니다.

### Q. 임금대장은 어떤 형식이 좋나요?

|------|--------|------|

| PDF (스캔본) | ⭐ 보통 | OCR로 글자 인식 후 AI 분석 |

- 손글씨나 흐린 문서는 정확도가 낮을 수 있습니다
- 추출 결과를 확인하고, 잘못된 부분이 있으면 참고만 하세요

### Q. "검토 필요"라고 나오면 어떻게 하나요?
