# 고용지원금 자동화 시스템 - 서비스화 추진 계획

## 현재 상태

- **버전**: MVP 완료
- **기능**: 문서 업로드 → OCR 추출 → 지원금 자격 분석 → 예상 수령액 계산
- **기술 스택**: React + Express + TypeScript + Tesseract.js

---

## Phase 0: 피드백 수집 (현재 단계)

### 목표
- 실제 사용자(노무사, HR 담당자)에게 공유하여 피드백 수집
- 기능 검증 및 UI/UX 개선점 파악

### 배포 방법: Replit

| 항목 | 내용 |
|------|------|
| 플랫폼 | Replit |
| 이유 | 코드 수정 없이 즉시 배포, URL 공유 용이, 실시간 수정 가능 |
| 비용 | 무료 (테스트용) |
| 예상 기간 | 1일 |

### 작업 내용
1. GitHub 연동 또는 코드 업로드
2. 환경 설정 (Node.js, npm install)
3. 빌드 및 실행
4. 공유 URL 생성 → 피드백 수집

### 수집할 피드백
- [ ] OCR 추출 정확도
- [ ] 지원금 계산 정확성
- [ ] UI/UX 사용성
- [ ] 추가 필요 기능
- [ ] 버그 리포트

## Phase 2: 웹 SaaS 출시 (구독 수익)

### 목표
- 웹 기반 SaaS로 구독 모델 운영
- 더 넓은 고객층 확보

### 기술 스택

| 항목 | 선택 | 비용 |
|------|------|------|
| 프론트엔드 | Vercel | 무료~$20/월 |
| 백엔드 | Railway 또는 Fly.io | $5~20/월 |
| 데이터베이스 | Supabase (PostgreSQL) | 무료~$25/월 |
| 인증 | NextAuth.js + 소셜 로그인 | 무료 |
| 결제 | 토스페이먼츠 | 수수료 2.8%~ |
| 파일 저장 | Cloudflare R2 | $0.015/GB |

### 시스템 구조

```
┌─────────────────────────────────────────────────────┐
│                    Vercel (프론트엔드)               │
│                    Next.js + React                  │
└─────────────────────┬───────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────┐
│                 Railway (백엔드 API)                 │
│                 Express + Node.js                   │
└──────────┬──────────────────────────┬───────────────┘
           │                          │
           ▼                          ▼
┌──────────────────┐      ┌───────────────────────────┐
│    Supabase      │      │     Cloudflare R2         │
│  - PostgreSQL    │      │   - 업로드 파일 저장       │
│  - Auth          │      │   - OCR 결과 캐시         │
└──────────────────┘      └───────────────────────────┘
```

### 가격 정책 (안)

| 플랜 | 가격 | 제한 |
|------|------|------|
| 무료 | 0원 | 월 3회 계산, 직원 10명 |
| 스타터 | 2.9만원/월 | 월 30회, 직원 50명 |
| 비즈니스 | 5.9만원/월 | 무제한 |
| 엔터프라이즈 | 협의 | 전용 인스턴스, SLA |
